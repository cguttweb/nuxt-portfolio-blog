Nuxt is a Vue-based framework for SSR or Server Side Rendering but it can also be used for static site generation.

### Routing

Routing is straightforward in Nuxt whereas in Vue apps you need to set the router and a routes file in Nuxt it is done via the creation of folders and files. Any pages created in the pages folders are automatically saved to the router config file which you can see in the .nuxt folder
